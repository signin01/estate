# RealEstate - Property Finder App

## Features
- Search properties by pincode
- Property comparison (up to 3 properties)
- Dark/Light mode toggle
- EMI Calculator
- Property Value Predictor (5 years)
- Rental Yield Calculator
- Neighborhood Safety Score
- Virtual Tour
- Price Drop Alerts
- Wishlist feature

## Tech Stack
- React.js
- React Router DOM
- Framer Motion

## Installation
1. Clone the repository
2. Run 
pm install
3. Run 
pm start
4. Open http://localhost:3000
